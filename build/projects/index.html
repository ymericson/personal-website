Some things I've worked on

data sciency work:

  • Created and integrated database models into a web app to display the Central Utility Plant's energy consumption data in an internal server. The C+ web app using shows an input form for NIH engineers to enter and search utility records, and visualizations using R, Qlik, and MS Report Builder summarise energy usage and cost avoidance. For NIH as a Civic Digital Fellow in summer 2020.
  • Identified Chicago neighborhoods where affordable housing is most likely to be an issue. Using open local government datasets and machine learning models, predicted areas with highest increases in price-to-income ratios in several years. Final project for CAPP 30254: Machine Learning for Public Policy. GitHub
  • Made a Django web app that outputs an energy source table and visual representations of state energy consumption trends. The state-level data from US EIA was imported via web scraping and API. Final project for CAPP 30122: Computer Science with Applications 2. Link GitHub
